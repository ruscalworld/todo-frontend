# Todo Frontend - Матрица Эйзенхауэра

Веб-приложение для управления задачами с использованием матрицы Эйзенхауэра. Автоматическое распределение задач по квадрантам на основе важности и срочности (дедлайна).

## Возможности

- **Аутентификация**: Регистрация и вход с JWT токеном
- **Управление задачами**: Создание, редактирование, удаление и завершение задач
- **Матрица Эйзенхауэра**: Визуализация задач в 4 квадрантах
  - Q1: Срочно и важно
  - Q2: Важно (не срочно)
  - Q3: Срочно (не важно)
  - Q4: Не срочно, не важно
- **Задачи на сегодня**: Фильтрация задач с близкими дедлайнами
- **Поиск**: Полнотекстовый поиск по задачам
- **Статистика**: Диаграммы распределения задач

## Запуск

### Режим разработки

```bash
npm run dev
```

### Сборка

```bash
npm run build
npm run start
```
